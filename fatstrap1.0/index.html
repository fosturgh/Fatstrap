We couldn’t decide between Foundation & Bootstrap so we combined them into the code masterclass.... Our framework is made up of other open source documents, we’ve just gone back to the drawing board with them and taken our favourite components and made them into this wonderful code buffet. Feel free to download Fatstrap and email with us or get in touch via skype if you’re having any problems with anything!

We’ve been a little greedy...

So we love various frameworks, and use them everyday, but we came up with the idea to combine them all! Fatsrap contains all the best bits from all the best frameworks, all squeezed together in this amaizing framework pie!

  • squeezed by the nailed it team

    We have gone through various frameworks dragging out the best bits, from the base css, to the javascript and jQuery functions. We’ve done all the hard work and broke it down into an easy and understandable bit of kit.

  • magento & wordpress unite

    No more need for mobile web based apps, this framework with make your site 100% mobile ready. Working on both Wordpress & Magneto this framework is more than ready to be spread onto your blog, eCommerce or portfolio site.

  • perfect portion for any device

    Suitable for anyone and everyone the new fatstrap can squeeze your chubby website into any device with ease! No overweight codes or heavyweight javascripts, simple to use, simple to intergrate and simply amazing.
